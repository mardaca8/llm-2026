```python
import pandas as pd
import matplotlib.pyplot as plt

# 1. Load the CSV file
df = pd.read_csv('sales.csv', parse_dates=['date'], index_col='date')

# Ensure data is sorted by date for rolling calculations
df.sort_index(inplace=True)

# 2. Calculate 30-day rolling average
df['rolling_avg'] = df['revenue'].rolling('30D', min_periods=1).mean()

# 3. Detect outliers (more than 2 std from mean)
mean_revenue = df['revenue'].mean()
std_revenue = df['revenue'].std()
outlier_threshold = 2 * std_revenue

# Create boolean mask for outliers
df['is_outlier'] = (df['revenue'] - mean_revenue).abs() > outlier_threshold

# 4. Plot the results
plt.figure(figsize=(12, 6))

# Plot raw revenue
plt.plot(df.index, df['revenue'], 
         label='Raw Revenue', 
         color='blue', 
         linewidth=1, 
         alpha=0.7)

# Plot rolling average
plt.plot(df.index, df['rolling_avg'], 
         label='30-Day Rolling Average', 
         color='green', 
         linewidth=2)

# Highlight outliers in red
outliers = df[df['is_outlier']]
plt.scatter(outliers.index, outliers['revenue'], 
            color='red', 
            s=50, 
            zorder=5, 
            label='Outliers',
            edgecolors='black')

# Add labels and formatting
plt.title('Revenue Analysis with 30-Day Rolling Average and Outliers', fontsize=14)
plt.xlabel('Date', fontsize=12)
plt.ylabel('Revenue', fontsize=12)
plt.legend()
plt.grid(True, alpha=0.3)
plt.tight_layout()

# Show statistics
print(f"Total data points: {len(df)}")
print(f"Outliers detected: {len(outliers)}")
print(f"Mean revenue: {mean_revenue:,.2f}")
print(f"Standard deviation: {std_revenue:,.2f}")
print(f"Outlier threshold: ±{outlier_threshold:,.2f}")

plt.show()
```

## Key Features:

1. **Data Loading**: 
   - Automatically parses the 'date' column as datetime and sets it as index
   - Sorts data chronologically (essential for time-based rolling windows)

2. **Rolling Average**:
   - Uses `rolling('30D')` for a true 30-day calendar window (not just 30 rows)
   - `min_periods=1` ensures calculation starts from first available data point

3. **Outlier Detection**:
   - Calculates overall mean and standard deviation of the entire revenue column
   - Flags points where absolute deviation > 2 standard deviations
   - Stores outlier flags in a new column for easy filtering

4. **Visualization**:
   - Raw revenue shown as thin blue line
   - Rolling average as thick green line
   - Outliers highlighted as large red dots with black edges
   - Includes grid, legend, and proper labeling
   - Displays summary statistics in console

## Important Notes:

1. **CSV Format Requirements**:
   - Must contain columns named `date` (datetime format) and `revenue` (numeric)
   - Example CSV structure:
     ```
     date,revenue,other_columns...
     2024-01-01,1500,...
     2024-01-02,1800,...
     ```

2. **Time Windows**:
   - `'30D'` means calendar days (30 days), not 30 business days
   - Requires pandas 0.19+ for offset-based rolling windows

3. **Outlier Definition**:
   - Uses **global** mean/std (entire dataset), not rolling statistics
   - For time-series with trends, consider using rolling statistics instead:
     ```python
     # Alternative: rolling-based outlier detection
     rolling_mean = df['revenue'].rolling('30D').mean()
     rolling_std = df['revenue'].rolling('30D').std()
     df['is_outlier'] = (df['revenue'] - rolling_mean).abs() > 2 * rolling_std
     ```

4. **Dependencies**:
   - Install required packages: `pip install pandas matplotlib`
